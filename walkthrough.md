# 경기 화면 UI 개선 및 선수 교체 시스템 리팩토링 결과 보고서

본 문서에서는 KBO 리그 시뮬레이터의 경기 화면 가독성 개선, UI 버그 수정 및 실시간 선수 교체 시스템(대타 기용, 포수 교체) 기능 추가에 대한 상세 변경 내역을 기록합니다.

## 1. 경기 화면 UI/UX 개선 사항

### 1.1. 타자 타율(AVG) 텍스트 상시 초록색 출력 버그 해결
* **기존 문제**: 타석의 타자가 삼진을 당하거나 안타를 기록하지 못했음에도, 배터 정보 카드 내 타율(AVG) 텍스트가 무조건 초록색으로 출력되던 오해의 소지가 있던 문제.
* **원인**: `index.html` 내 `#b-avg` 엘리먼트에 `style="color:var(--accent3)"` 스타일이 하드코딩되어 고정되어 있었음.
* **해결**: 인라인 스타일 코드를 제거하여 기본 텍스트 색상(흰색 계열)으로 환원시켰으며, 다른 스탯들(HR, RBI, OPS)과 일관되게 정돈했습니다.

### 1.2. 경기 종료 화면 MVP 선정 기준 설명글 제거
* **변경**: 오늘의 MVP 카드 하단에 노출되던 복잡한 줄글(가중치 설명 텍스트)을 제거하여 경기 종료 연출 화면을 한층 간결하고 직관적으로 보이게 개선했습니다.

### 1.3. '오늘의 성적' UI 2분할 레이아웃 개편
* **기존**: 타자의 타석 기록(타석, 안타, 홈런, 삼진 등)만 단일 열로 단순 출력되고 있었음.
* **수정**:
  * 상단의 '오늘 성적' 타이틀 텍스트를 제거하여 컴팩트화.
  * **좌측(투수)** 및 **우측(타자)**의 2분할 레이아웃으로 변경.
  * 우측 타자 기록은 기존과 동일하게 유지하고, **좌측 투수 기록(당일 투구 수, 소화 이닝, 실점/자책점, 삼진, 피안타, 사사구)**을 실시간 연동하여 표시하도록 `updateTodayStats` 함수를 개편했습니다.

### 1.4. 선수 정보 카드 헤더 가로선 1줄 정렬 및 좌우 대칭 배치
* **변경**:
  * 선수 이름 아래 배치되어 가독성을 떨어뜨리던 투구 수 정보를 제거하고 수평 배치를 통해 컴팩트한 레이아웃을 구현했습니다.
  * 투수/타자의 이름과 정보(`[유리] 우투`, `우타 [유리]`)를 동일한 가로 높이(baseline)에 한 줄로 정렬하였습니다.
  * 투수는 가장 바깥쪽인 맨 왼쪽에 `[유리]` 뱃지가 정렬되고, 타자는 맨 오른쪽에 `[유리]` 뱃지가 오도록 완벽한 데칼코마니(좌우 대칭) 정렬을 완료했습니다.

## 2. 경기 중 실시간 선수 교체(대타/포수) 시스템 고도화

기존에 투수 교체 기능만 제한적으로 지원되던 선수 교체 모달을 야구 룰과 매치업 상황에 유동적으로 반응하도록 고도화하였습니다.

### 2.1. 상황별 교체 리스트 노출
* **수비 시 (Defense)**: 투수 교체 및 **포수 교체** 영역만 활성화됩니다.
  * 벤치 타자 중 수비 포지션이 `포수`인 선수가 최상단에 우선 정렬되며, 주황색 `[포수]` 뱃지가 부착되어 한눈에 식별할 수 있습니다.
  * 포수 교체 진행 시, 해당 선수는 라인업에 `pos: 'C'` 상태로 수비진에 배치됩니다.
* **공격 시 (Offense)**: 투수 교체 영역이 감추어지고 **대타 기용** 영역만 활성화됩니다.
  * 벤치 타자 리스트에서 선수를 골라 현재 타석에 선 타자와 즉각 교체할 수 있습니다.

### 2.2. 야구 룰에 따른 엄격한 자원 관리
* 경기 중 라인업에서 빠진 선수는 **`usedToday = true`**로 설정되어, 다음 교체 창의 벤치 멤버 목록에 다시 나타나지 않아 중복 출전이 불가능합니다.

### 2.3. 실시간 타석 상태(PA) 및 UI 연동
* **0구 시 교체 연동**: 볼카운트가 진행되기 전(0구 상황)에 대타나 투수 교체를 수행하면, 교체된 선수의 데이터(투수의 구질지수, 타자의 안타 확률 등)를 반영하여 **해당 타석의 결과(PR) 및 투구 시퀀스(seq)를 실시간으로 재계산**하도록 연동했습니다.
* **중간 투구 승계**: 투구가 진행 중인 카운트 상태에서 대타 기용 시에는, 기존 진행 중이던 카운트를 올바르게 계승하여 경기가 진행됩니다.
* **UI 즉시 업데이트**: 교체 완료 후 전광판 UI(`updateGameUI`), 라인업 리스트(`updateLnpUI`), 투수/타자 정보 카드가 딜레이 없이 실시간으로 리프레시됩니다.
